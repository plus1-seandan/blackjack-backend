# Blackjack - Server

Backend of the blackjack game app.

## Tech Stack

Tech Stack:
Node.js <br/>
Express <br/>
Redis <br/>
Postgres <br/>

Deployed through:
AWS <br/>
Surge <br/>
Docker <br/>
